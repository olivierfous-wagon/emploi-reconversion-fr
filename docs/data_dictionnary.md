## INSEE — Enquête emploi en continu (EEC)

- ANNEE : année de référence de l’enquête
- TRIM : trimestre de référence
- periode : date de début du trimestre (créée)
- AGE5 : tranche d’âge (6 classes)
- SEXE : sexe
- STAT_CS : statut d’activité déclaré
- STAT_CSPRL : statut d’activité selon catégorie socioprofessionnelle
